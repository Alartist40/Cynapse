# Voice Data Storage Directory

This directory stores voice recognizer data:
- Whisper.cpp models
- Voice profiles
- Audio samples for testing

**Note**: Download Whisper models from:
https://huggingface.co/ggerganov/whisper.cpp

Example:
```bash
wget https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-tiny.en-q5_1.bin
```
